# Anycubic Kobra S1 — Maintenance Guide

This repository contains a compact, user-friendly maintenance guide for the **Anycubic Kobra S1** / **Kobra S1 Combo** 3D printer, generated as a single HTML file (`maintenance.html`).

## Background

My printer informed me (after 500 hours of printing) that I should perform maintenance. I checked the [Anycubic Wiki maintenance page](https://wiki.anycubic.com/en/fdm-3d-printer/kobra-s1-combo/maintenance-recommendations) for guidance, but found it a bit rough — the information is there, but the layout, formatting, and completeness weren't what I wanted for a reference I'd use regularly.

So I asked **Deep Seek AI** to help me put together something nicer: a single-page HTML guide that's clean, step-by-step, printer-friendly, and pulls in common-sense FDM best practices beyond what the wiki covers. The AI (assisting as the agent "99") surveyed the wiki, searched for supplementary maintenance knowledge, and iterated with me on layout, content, and presentation until we had something worth sharing.

## What's Inside

- **Quick-reference schedule** — tasks grouped by frequency (every print, weekly, monthly, quarterly, per-spool)
- **9 maintenance sections** — Z-screw, Y-rods, X-rods, camera, cutter, hotend/nozzle, silicone sock, fans, spring steel plate
- **Safety-first** guidelines
- **Replacement parts table** — SKUs, prices, and direct links to the Anycubic Official Store, Amazon, Micro-Swiss, and 3DJake
- **Bonus tips** — belts, PTFE tube, filament storage, PID tuning, maintenance logging
- **Pre-print walkaround checklist**
- **Extruder disassembly video** — linked via thumbnail to YouTube
